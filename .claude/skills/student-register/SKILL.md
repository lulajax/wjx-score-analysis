# 新学员信息登记

将学员报名信息追加到 `新学员登记表.xlsx`。

## 使用流程

1. 从用户消息中提取以下字段（缺失的字段留空即可）：

| 序号 | 字段 | 示例 |
|------|------|------|
| 1 | 协议是否签署 | 是/否 |
| 2 | 学员姓名 | 小红 |
| 3 | 性别 | 女 |
| 4 | 年龄 | 24岁 |
| 5 | 专业 | 美术与书法 |
| 6 | 学历和学位 | 研究生 |
| 7 | 学校 | xxx |
| 8 | 从事何种行业 | 在校学生 |
| 9 | 备考地区 | 浙江、江西 |
| 10 | 目标考试 | 省考、事业单位、央企国企 |
| 11 | 报考年份 | 2027年 |
| 12 | 参考经历 | 0基础 |
| 13 | 每日学习时长 | 3-6个小时 |
| 14 | 政治面貌 | 群众 |
| 15 | 毕业时间 | 2027年6月 |
| 16 | 邮寄地址 | 浙江省杭州市... |
| 17 | 电话 | 15216123456 |
| 18 | 报名课程产品名称 | 院长伴学双向入编定制私教直播 |
| 19 | 实付金额 | 8000 |
| 20 | 业绩归属 | xxx |
| 21 | 报班时间 | 2026年4月15日 |
| 22 | 学员特殊情况备注 | （可为空） |

2. 将提取的字段组成 JSON，运行脚本：

```bash
python3 /home/lujunjie/Project/wjx-score-analysis/.claude/skills/student-register/register_student.py \
  --xlsx /home/lujunjie/Project/wjx-score-analysis/新学员登记表.xlsx \
  --data '{"学员姓名":"小红","性别":"女","年龄":"24岁","专业":"美术与书法","学历和学位":"研究生","学校":"xxx","从事何种行业":"在校学生","备考地区":"浙江、江西","目标考试":"省考、事业单位、央企国企","报考年份":"2027年","参考经历":"0基础","每日学习时长":"3-6个小时","政治面貌":"群众","毕业时间":"2027年6月","邮寄地址":"浙江省杭州市浙江财经大学下沙校区桃李苑6号菜鸟驿站","电话":"15216123456","报名课程产品名称":"院长伴学双向入编定制私教直播","实付金额":"8000","业绩归属":"xxx","报班时间":"2026年4月15日","学员特殊情况备注":"","协议是否签署":"是"}'
```

**用户说"登记新学员"或提供学员信息时，提取字段后直接运行上面的命令。**

## 字段说明

- 脚本自动匹配 Excel 表头，如果表头不存在会自动新增列
- `报班时间` 和 `毕业时间` 会自动解析为日期格式
- `实付金额` 会自动转为数字
- `电话` 保持文本格式
- 数据追加到第一个空行，不会覆盖已有数据
