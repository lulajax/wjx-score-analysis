# 新学员信息登记

将学员报名信息追加到 `新学员登记表.xlsx`。

## 使用流程

1. 从用户消息中提取以下字段（缺失的字段留空即可）：

| 序号 | 字段 | 示例 |
|------|------|------|
| 1 | 合同是否签署 | 是/否 |
| 2 | 入学时间 | 2026年4月15日 |
| 3 | 学员姓名 | 小红 |
| 4 | 报考年份 | 2027年 |
| 5 | 报名课程产品名称 | 院长伴学双向入编定制私教直播 |
| 6 | 联系方式 | 15216123456 |
| 7 | 实付金额 | 8000 |
| 8 | 性别 | 女 |
| 9 | 年龄 | 24岁 |
| 10 | 学历 | 研究生 |
| 11 | 学校 | xxx |
| 12 | 毕业时间 | 2027年6月 |
| 13 | 专业 | 美术与书法 |
| 14 | 所考省份 | 浙江、江西 |
| 15 | 目标考试 | 省考、事业单位、央企国企 |
| 16 | 参考经历 | 0基础 / 备考3月 / 备考2年 |
| 17 | 收货地址 | 浙江省杭州市... |
| 18 | 从事何种行业 | 在校 / 在职3年 |
| 19 | 备注 | （可为空） |

2. 将提取的字段组成 JSON，运行脚本：

```bash
python C:/Users/PC-749/Project/wjx-score-analysis/.claude/skills/student-register/register_student.py \
  --xlsx C:/Users/PC-749/Project/wjx-score-analysis/新学员登记表.xlsx \
  --data '{"合同是否签署":"是","入学时间":"2026年4月15日","学员姓名":"小红","报考年份":"2027年","报名课程产品名称":"院长伴学双向入编定制私教直播","联系方式":"15216123456","实付金额":"8000","性别":"女","年龄":"24岁","学历":"研究生","学校":"xxx","毕业时间":"2027年6月","专业":"美术与书法","所考省份":"浙江、江西","目标考试":"省考、事业单位、央企国企","参考经历":"0基础","收货地址":"浙江省杭州市浙江财经大学下沙校区桃李苑6号菜鸟驿站","从事何种行业":"在校学生","备注":""}'
```

**用户说"登记新学员"或提供学员信息时，提取字段后直接运行上面的命令。**

## 字段说明

- 脚本自动匹配 Excel 表头，如果表头不存在会自动新增列
- `入学时间` 和 `毕业时间` 会自动解析为日期格式
- `实付金额` 会自动转为数字
- `联系方式` 保持文本格式
- 数据追加到第一个空行，不会覆盖已有数据
